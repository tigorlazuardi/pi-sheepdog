# Reference

## Commands

- `/sheepdog`
- `/sheepdog config`

## Paths

- `${agentDir}/sheepdog/config.json`
- `${agentDir}/.cache/sheepdog/state.json`
- `${agentDir}/sheepdog/debug.log`

## Adapter ids

- `generic`
- `anthropic`
- `openai-compatible`

## State statuses

- `pending`
- `fired`
- `cancelled`
- `expired`

## Privacy and redaction

Expected redactions cover:

- auth headers
- cookies
- tokens
- API keys
- JWTs
- passwords
- private keys
- credential contents
- sensitive adapter args

## Legacy provenance

It is legacy provenance only and not the current consumer contract.
